los input son los elemnetos que se crean como contenido de un formulario, y nos permite interactuar con el usuario, recolectar informacion atrabves de nuestra pagina

TAREA AVERIGUAR LOS TIPOS DE INPUT

  • Texto (text) Permite escribir texto libre.
  • Contraseña (password) Oculta el texto ingresado.
  • Correo electrónico (email) Valida que el contenido tenga formato de email.
  • Número (number) Solo permite valores numéricos.
  • Fecha (date) Muestra un selector de fecha.
  • Color (color) Muestra un selector de color.
  • Archivo (file) Permite subir archivos.
  • Restablecer (reset) Borra los valores ingresados en el formulario.
  • Imagen (image) Usa una imagen como botón de envío.
